# 观澜 - 股票知识学习与演练平台

## Overview
集知识库学习、量化回测、模拟交易、技术指标分析于一体的股票投资学习平台

## Tech Stack
Python / JavaScript

## Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Start development server
python app.py

# Run tests
npm run test  # or equivalent
```
